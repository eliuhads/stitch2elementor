GUÍA DE CONFIGURACIÓN DE MCP SERVERS PARA STITCH → ELEMENTOR
============================================================

1. INSTALACIÓN DE SERVIDORES GLOBALES (Obligatorio)
Evita timeouts por compilación in-time mediante:
> npm install -g wp-elementor-mcp
> npm install -g elementor-mcp

2. GESTIÓN DE ACCESOS REST
- WordPress: Solicita/Crea una "Application Password" oficial del panel wp-admin (Perfil > Application Passwords). Formato estricto a respetar: "xxxx xxxx xxxx xxxx xxxx xxxx".
- Google Stitch: Emplea Google Cloud Console > "Credentials" > "API key".

3. FRAGMENTO TARGET PARA `mcp_config.json`
Edita MANUALMENTE el archivo `mcp_config.json` e inserta los bloques. ADVERTENCIA: Apaga variables paralelas innecesarias de MCP (Puppeteer, Figma, etc) configurando "disabled": true para evitar dispersión del LLM.

"StitchMCP": {
  "command": "npx",
  "args": ["-y", "mcp-remote", "https://stitch.googleapis.com/mcp", "--header", "X-Goog-Api-Key: TU_API_KEY"],
  "disabled": false
},
"wp-elementor-mcp": {
  "command": "wp-elementor-mcp",
  "args": [],
  "env": {
    "WORDPRESS_BASE_URL": "https://tu-dominio.com",
    "WORDPRESS_USERNAME": "TU_USUARIO_ADMIN",
    "WORDPRESS_APPLICATION_PASSWORD": "xxxx xxxx xxxx xxxx xxxx xxxx",
    "WP_MCP_MODE": "advanced"
  },
  "disabled": false
},
"elementor-mcp": {
  "command": "elementor-mcp",
  "args": [],
  "env": {
    "WP_URL": "https://tu-dominio.com",
    "WP_APP_USER": "TU_USUARIO_ADMIN",
    "WP_APP_PASSWORD": "xxxx xxxx xxxx xxxx xxxx xxxx"
  },
  "disabled": false
}
